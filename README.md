
# YouTube Spam Comment Classification using Machine Learning & NLP

This repository contains a comprehensive research-focused data science project dedicated to identifying and classifying spam comments across YouTube videos. Using Natural Language Processing (NLP) text-preprocessing methods and advanced machine learning modeling, the project addresses the growing problem of automated and malicious comment spam.
